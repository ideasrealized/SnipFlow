SnipFlow Edge Hover Zones

The highlighted areas show where the edge hover trigger zones are located.

Default settings:

  • Position: left-center
  • Trigger size: 50px
  • Delay: 200ms
To activate overlay: Move your mouse to one of the highlighted zones and wait 200ms

If the overlay doesn't appear, check:

  1. Edge hover is enabled in settings
  2. The overlay window is created (check dev tools)
  3. No JavaScript errors in console
